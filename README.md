## Omnicam

**Omnicam** is a browser-based camera application built with modern web technologies.

Built with TypeScript (95+% of the codebase)

## Use locally

1. Clone the repository

    ```bash
    git clone https://github.com/iamovi/omnicam.git
    ```

2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`

After runing `npm run dev` in your terminal, you will see a message like this:

```bash
> omnicam@1.0.0 dev
> vite


  VITE v7.3.1  ready in 771 ms

  ➜  Local:   http://localhost:8080/
  ➜  Network: http://<your-local-ip>:8080/
  ➜  press h + enter to show help
```

Now, Open your browser and go to:

```bash
http://localhost:8080/
```

### Usage

Use normally or with keyboard shortcuts:

- Press `space` to take a picture
- Hold `space` to record a video
- Press `s` to switch between front and back camera
